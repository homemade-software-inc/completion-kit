Hi Peter,

I just released CompletionKit 0.1.0, an open-source Rails engine for prompt regression testing. It mounts in any Rails app (or runs as a bundled standalone app) and provides a UI, REST API, and MCP server for testing GenAI prompts against real datasets, scoring outputs against custom rubrics with an LLM-as-judge, and tracking prompt versions.

It supports OpenAI, Anthropic, Ollama (any OpenAI-compatible local endpoint), and 100+ models via OpenRouter -- none of the existing prompt testing tools support that combination at the same time.
